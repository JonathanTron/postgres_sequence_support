= Postgres Sequence Support

Very basic support for manipulating PostgreSQL sequences from ActiveRecord.

== Examples

  User.next_id              # => 1
  User.current_id           # => 1

  c = ActiveRecord::Base.connection
  c.set_sequence('users_id_seq', 1000)
  c.next_in_sequence('users_id_seq')      # => 1001
